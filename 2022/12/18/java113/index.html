java | 生产者/消费者

发表于 2022-12-18 分类于 java ， 进阶 ， 多线程 ， 设计模式 ， 异步模式 ， 生产者-消费者 阅读次数： Valine：

这里说一下，生产者-消费者和 保护性暂停 的区别。

  • 与保护性暂停不同，不需要产生结果和消费结果一一对应
  • 消费队列可以平衡生产和消费的线程资源
  • 生产者仅负责产生结果，消费者仅负责处理结果
  • 消息队列是有容量限制的，满时不会再加入数据，空时不会再消耗数据
  • JDK 中各种堵塞队列，采用该模式
  • 异步的意思是，消息产生了，会先放在队列中，可能不会立刻消费

redis | 事务与锁

发表于 2022-12-04 分类于 sql ， redis ， 基础 阅读次数： Valine：

讲一下事务和锁。

事务

一组命令的集合，一块执行。

所有的事务中的命令都会被序列化，在事务的执行过程中，会被顺序执行，并且，一次性，顺序性的执行全部命令。

  • 原子性
    • redis 单条命令是保存原子性的，但是，事务不保证原子性
  • 没有隔离级别的概念
    • redis 的一组命令，进入队列后，并不是马上执行，而是，主动执行的时候才执行
      • 队列 set set set 执行

事务过程

  • 开启事务(multi)
  • 命令入队
  • 执行事务(exec)

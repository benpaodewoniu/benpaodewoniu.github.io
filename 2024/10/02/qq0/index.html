设置

默认对外是 6099 端口，所以，你的服务器也要开启这个端口的访问。

使用 docker exec -it napcat /bin/bash 进入运行的容器中。

在 /app/napcat/config 中 webui.json 记录了登录用的 token。

访问 http://IP:6099/webui/login.html，输入 token 就进入了配置界面。

每次重新启动 docker ，这个 token 都会变。

catattacknft 是 base 测试网的一个 NFT 小游戏。

  • catattacknft

Base 测试网是建立在 Goerli 上面的「应该是二层，不过，我没去深究」。这里假设你已经把 Goerli 上的 ETH 跨链到了 Base-Goerli。

先说一下 catattacknft 的游戏规则

  • catattacknft 是 ERC1155 ，其 ID 有 4 类
    • 0: 基础小猫
    • 1: 燃烧小猫
    • 2: 忍者小猫
    • 3: 忍者小猫杀死基础小猫后，会变成 3
  • 具体规则
    • 免费 mint 一个基础小猫
    • 基础小猫通过 transfer 到另外一个账户，该账户会获得一个燃烧小猫
    • 燃烧小猫通过 burn 会获得一个忍者小猫
    • 忍者小猫可以杀死基础小猫
      • 忍者小猫可以一直杀下去
  • 如果你地址中有了忍者猫
    • 只能杀死别人地址的基础猫
    • 不能 mint 其他猫了

这里可能有一个奇怪的现象。

我明明 mint 了一个基础小猫，为什么无法转移？

这是因为，现在区块链上，已经有大量的科学家时刻监控链上行为，一旦有基础小猫出现，他就会用忍者小猫杀死基础小猫。「可能，科学家们想让他们手里的忍者小猫变得稀有」

图中红框处是我转移了一个基础小猫，而绿框处是别人对我的基础小猫进行了狙杀。

并且，其狙杀行为和我转移小猫处于同一区块，这应该是监控了 penging 进行专门的狙杀。

所以，手动的话，除非非常幸运，不然，很难获得忍者小猫。

撸羊毛的正确姿势

发表于 2021-04-05 更新于 2021-05-16 分类于 奇淫巧技 ， 基础 阅读次数： Valine：

在币圈中，撸空投是一个白嫖暴富的途径，这里教大家怎么来撸空投。

币圈发展到现在防止脚本撸的方法已经很完善了，一般来说，想要撸空投需要单钱包地址对应一个单 IP 还对应一个单硬件地址。

另外，正确的撸空投姿势应该是脚本撸，而不是手动。

我将从以下几个方面讲一下如何撸空投。

  • 批量生成钱包地址
  • 无限邮箱
  • 手机验证码
  • IP 池
  • 硬件地址替换
  • 脚本编写

批量生成地址

这里主要用的是一个非常牛逼的网站。

  • cointool

这个网站有批量生成地址的功能，非常强大。

无限邮箱

参考资料如下

  • 教你一招：轻松获取无限个邮箱小号

我这里主要是用了教程中的第二招。「傲游：百变邮箱」。

我先介绍一下这个邮箱的功能，你就知道有多强大了。先绑定一个真实的邮箱地址，然后，自己根据官网要求创建一个域名，你就可以在这个域名下，生成无限多的影子邮箱。

什么是影子邮箱呢？影子邮箱有一个独立的邮箱地址，在遨游的影子邮箱，具备某平台向该邮箱发送邮件，然后，该影子地址会自动把邮箱转发到你绑定的邮箱，非常 nice。

另外，如果你想要接受影子邮箱的转发邮件，需要在你绑定的那个邮箱平台创建域名白名单，这个在遨游官网有非常详尽的指导，这里不再说明。

另外，上面参考资料还有一种方法很有意思，建议看一下。
